Chào mừng đến với Git!

Đây là nội dung mẫu để bạn test commit, restore, và các thao tác Git khác.

  • Bước 1: Sửa file này và lưu lại
  • Bước 2: Chạy git add test1.html
  • Bước 3: Chạy git commit -m "Nội dung thay đổi lần 2"
  • Bước 4: Chạy git log để xem lịch sử commit
  • Bước 5: Thử git checkout test1.html để khôi phục
  • Bước 6: Thêm dòng này để tạo commit mới và quan sát trên Graph, tôi thêm từ local để quan sát
  • Bước 7: Sử dụng git log --oneline --graph --all để xem lịch sử dạng cây
  • Bước 8: Cài extension Git Graph trên VS Code để xem trực quan
  • Bước 9: Đẩy commit lên GitHub bằng git push origin feature-test1
